🔗 Live Project : https://future-fs-01-portfolio-lake.vercel.app/

I developed a fully responsive Personal Portfolio Website to showcase my skills, projects, and contact information.


🔧 Tech Stack Used:

 ✨ Frontend: HTML, CSS, JavaScript (with smooth animations & responsive design)

 ✨ Backend: Node.js + Express

 ✨ Database: MongoDB (used for storing contact form submissions)


📌 Key Features:

 ✔ Modern UI with animations

 ✔ Project section with GitHub links and previews

 ✔ Contact form connected to backend & database


This project helped me strengthen both my frontend and backend development skills and build a professional platform to present my work.
